# 文章转漫画 - 使用示例

## 示例 1：短篇散文

### 输入文章

> 清晨，小雪推开了咖啡馆的门。阳光透过落地窗洒在木质地板上，空气中弥漫着现磨咖啡的香气。她在靠窗的位置坐下，翻开随身携带的笔记本，开始记录昨晚的梦境。
> 
> "一杯热拿铁，谢谢。"她对走过来的店员微笑着说。
> 
> 店员是个戴着眼镜的年轻男生，腼腆地点了点头："好的，请稍等。"
> 
> 小雪望向窗外，街道上行人匆匆，梧桐树的叶子已经开始泛黄。她觉得这个秋天的早晨，美好得让人想哭。

### 角色档案

```
角色 1 - 小雪:
  年龄外观: 22岁左右的年轻女性
  发型发色: 黑色齐肩微卷短发，整齐的齐刘海
  面部特征: 圆脸，弯弯的杏眼，小巧的鼻子，温柔的微笑
  体型: 纤细中等身材
  标志性服装: 米白色高领毛衣，浅蓝色牛仔裤
  标志性配饰: 柔软的红色围巾
  气质关键词: 温柔文静，文艺气质

角色 2 - 店员小哥:
  年龄外观: 20岁左右的年轻男性
  发型发色: 深棕色短寸头，干净利落
  面部特征: 瘦长脸型，单眼皮，腼腆的笑容
  体型: 偏瘦
  标志性服装: 深蓝色围裙配白色T恤，胸前有咖啡店logo
  标志性配饰: 圆框黑边眼镜
  气质关键词: 腼腆内向，认真专注
```

## 示例 2：叙事片段

### 输入文章

> 老陈每天傍晚都会去公园的长椅上坐着。他的金毛犬阿黄总是乖乖地蹲在旁边，陪他看夕阳。这天，一个小女孩跑过来，手里拿着一朵蒲公英。"爷爷，送给你！"老陈愣了一下，然后笑了，眼角的皱纹像花开一样舒展开来。

### 角色档案

```
角色 1 - 老陈:
  年龄外观: 65岁左右的老年男性
  发型发色: 花白短发，略有些稀疏
  面部特征: 方正脸型，深深的法令纹，慈祥的笑眼，眼角有明显的皱纹
  体型: 中等偏壮，微微驼背
  标志性服装: 灰色格子衬衫，深棕色休闲裤
  标志性配饰: 手腕上的老旧电子表
  气质关键词: 慈祥温厚，沉稳安静

角色 2 - 阿黄（金毛犬）:
  外观: 成年金毛寻回犬，毛色金黄柔顺，表情友善温顺，脖子上系着红色项圈

角色 3 - 小女孩:
  年龄外观: 5岁左右
  发型发色: 黑色双马尾辫，扎着粉色蝴蝶结
  面部特征: 圆嘟嘟的脸蛋，大大的黑眼睛，缺了一颗门牙的笑容
  体型: 矮小可爱
  标志性服装: 鹅黄色连衣裙，白色小凉鞋
  标志性配饰: 无
  气质关键词: 天真活泼，甜美可爱
```
